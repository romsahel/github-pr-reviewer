Keyboard Shortcuts

Key Action
r Toggle last-hovered line as reviewed
Shift + N Jump to next unreviewed line
Shift + P Jump to previous unreviewed line
Alt + R Toggle reviewed (browser command)
Shift + Alt + N Next unreviewed (browser command)
Shift + Alt + P Previous unreviewed (browser command)

Browser commands can be customized at about:addons → Manage Extension Shortcuts. In-page shortcuts (r, Shift+N, Shift+P) work automatically on any GitHub PR files page.
